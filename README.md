Data Cleaner Tool
📌 Overview
Data Cleaner Tool is a Python-based utility designed to clean and preprocess CSV datasets efficiently.
The tool helps users prepare raw datasets for analysis by automating common data-cleaning tasks such as removing duplicate records, handling missing values, standardizing data, and exporting the cleaned output into a new CSV file.
This project demonstrates practical Python, Pandas, file handling, and data preprocessing skills.
🎯 Project Objective
To build a simple command-line data cleaning tool that allows users to:
Load raw CSV datasets
Detect and clean data quality issues
Remove duplicate records
Handle missing values
Export cleaned datasets for further analysis

🚀 Installation
Clone the repository:
git clone https://github.com/kontamhrushikesh/Data-Cleaner-Tool.git
Navigate to the project folder:
cd Data-Cleaner-Tool
Install the required dependency:
pip install pandas
▶️ Usage
Run the tool:
python Data_cleaner.py
When prompted, enter the path of your CSV file:
Enter the path to your CSV file:

After processing, enter the path where you want to save the cleaned file:
Enter path to save cleaned CSV file:
